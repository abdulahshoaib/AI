Informed search algorithms (also called heuristic search algorithms) are search techniques
that use additional problem-specific knowledge to guide the search toward the goal more efficiently.

They make use of:
* Initial State
* Goal State
* Actions available between states
* Heuristic information

# Heutistic
A heuristic is a rule of thumb or practical strategy used in informed search to guide problem solving toward a goal more efficiently.
Instead of checking every possible path, a heuristic estimates how close a current state is to the desired solution, helping the
algorithm prioritize more promising options.
