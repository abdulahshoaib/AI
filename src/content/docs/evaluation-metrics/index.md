---
title: Evaluation Metrics
description: Overview of model evaluation metrics for regression and classification tasks.
---
Evaluation metrics measure how well a machine learning model performs on data that was not used during training.

The correct metric depends on the prediction task:

- Regression metrics evaluate continuous numerical predictions.
- Classification metrics evaluate predicted categories or class labels.

## Regression Metrics

Regression metrics compare predicted numerical values with actual numerical values. Common examples include:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R-squared score

## Classification Metrics

Classification metrics compare predicted classes with actual classes. Common examples include:

- Confusion matrix
- Accuracy
- Precision
- Recall
- F1 score
- ROC-AUC
- Classification report
